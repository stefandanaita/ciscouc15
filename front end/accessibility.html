Accessibility


Accesskeys for ABC Design


Accesskeys or access keys provide you the option of moving to a specific part of the web page via keyboard, without using the mouse. Depending on the web browser you can use the following key combination:


Web browser Home - H About - M News - N Demo - D Accessibility - A Contact - C
Google Chrome Alt + H Alt + M Alt + N Alt + D Alt + A Alt + C
Firefox(Windows) Shift + Alt + H Shift + Alt + M Shift + Alt + N Shift + Alt + D Shift + Alt + A Shift + Alt + C
Firefox(Mac) Ctrl + H Ctrl + M Ctrl + N Ctrl + D Ctrl + A Ctrl + C
Safari Ctrl + Alt + H Ctrl + Alt + M Ctrl + Alt + N Ctrl + Alt + D Ctrl + Alt + A Ctrl + Alt + C
Internet Explorer Alt + H
